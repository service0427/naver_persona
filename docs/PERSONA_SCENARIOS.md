# 페르소나 시나리오 정의서

> 작성일: 2025-12-29
> 목적: 쇼핑 검색 전 자연스러운 행동 패턴 학습

---

## 1. 개요

### 1.1 최종 목표
```
┌─────────────────────────────────────────────────────────────┐
│  최종 타겟: 네이버 쇼핑에서 특정 상품 검색                    │
│                                                             │
│  문제: 갑자기 쇼핑 검색만 하면 봇으로 의심                   │
│  해결: 페르소나에 맞는 서비스 이용 이력 쌓기                 │
│                                                             │
│  예: 30대 남성 직장인이 '무선 마우스'를 검색하려면           │
│      → IT 뉴스 읽기, 전자기기 카페 방문, 관련 블로그 등      │
│      → 자연스러운 관심사 흐름 형성                          │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 네이버 서비스 맵

| 서비스 | URL | 용도 | 연령대 |
|--------|-----|------|--------|
| 뉴스 | news.naver.com | 시사, 관심사 학습 | 전 연령 |
| 블로그 | blog.naver.com | 제품 리뷰, 정보 탐색 | 전 연령 |
| 카페 | cafe.naver.com | 커뮤니티, 관심사 | 20~40대 |
| 지식iN | kin.naver.com | 제품 문의, 비교 | 전 연령 |
| 스포츠 | sports.naver.com | 스포츠 뉴스 | 20~40대 남성 |
| 연예 | entertain.naver.com | 연예 뉴스 | 20~30대 |
| 웹툰 | comic.naver.com | 웹툰 | 10~30대 |
| 시리즈 | series.naver.com | 웹소설 | 20~40대 |
| 부동산 | land.naver.com | 부동산 정보 | 30~50대 |
| 증권 | finance.naver.com | 주식, 금융 | 30~50대 |
| 쇼핑 | shopping.naver.com | **최종 타겟** | 전 연령 |

│  └─ 관련 유튜브 영상 (네이버 검색 경유)                      │

## 5. 검색어 흐름 패턴

### 5.1 자연스러운 검색어 진화

```
┌─────────────────────────────────────────────────────────────┐
│  검색어 진화 패턴 (예: 무선 이어폰 구매)                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [1단계] 문제 인식                                          │
│  └─ "이어폰 한쪽 안들림" / "유선 이어폰 단선"               │
│                                                             │
│  [2단계] 정보 탐색                                          │
│  └─ "무선 이어폰 추천" / "에어팟 vs 갤럭시버즈"             │
│                                                             │
│  [3단계] 비교 검토                                          │
│  └─ "갤럭시버즈2 프로 후기" / "갤럭시버즈 노이즈캔슬링"     │
│                                                             │
│  [4단계] 구매 결정                                          │
│  └─ 쇼핑: "갤럭시버즈2 프로" / "갤럭시버즈2 프로 최저가"    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 페르소나별 검색어 스타일

| 페르소나 | 검색어 스타일 | 예시 |
|----------|--------------|------|
| 20대 남성 | 줄임말, 신조어 | "겜용 마우스", "가성비 키보드" |
| 20대 여성 | 트렌드 용어 | "요즘 쿠션", "인생템 립스틱" |
| 30대 직장인 | 구체적, 비교형 | "삼성 vs LG TV", "2025 신형 에어컨" |
| 30-40대 주부 | 실용적, 후기 중심 | "아기 있는 집 청소기", "후기 좋은 밥솥" |
| 50대+ | 정자체, 구체적 | "무릎 좋은 등산화", "허리 편한 의자" |
